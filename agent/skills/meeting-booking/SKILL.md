# Meeting and booking

## Tools

Search connection_search with: "meeting", "booking", "cuoc hop", "dat phong"

Key tools:
- `meeting_list_meetings` — list upcoming meetings
- `meeting_list_groups` — list meeting room groups
- `meeting_list_repeated_meetings` — list recurring meetings
- `booking_list_bookings` — list room bookings
- `booking_list_groups` — list booking groups
- `booking_list_resources` — list bookable resources
- `booking_list_purposes` — list booking purposes

## Reading schedule

- "Lich hop tuan nay?" → list meetings, filter by date range
- "Phong hop nao con trong?" → list bookings, find gaps

## Note

The Base.vn Meeting and Booking APIs are read-only via MCP. Creating meetings
or booking rooms is not supported. If the user asks to schedule, tell them to
do it directly in Base.vn.
